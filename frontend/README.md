# MITTI MITRA - Frontend

## Overview
The frontend is built using React (Vite) and Tailwind CSS. It provides a mobile-first interface for farmers to input soil data and view crop recommendations.

## Structure
- `src/components`: Reusable UI components (Navbar, SoilForm, Results).
- `src/pages`: Main pages (Home, Dashboard).
- `src/services`: API integration logic.

## Running Locally
1. `npm install`
2. `npm run dev`
